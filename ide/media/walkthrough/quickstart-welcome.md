# Welcome to NiceGeo IDE

NiceGeo IDE provides a focused editing experience for **`.ncg`** files in VS Code.

## What the extension includes

- **Language feedback**: diagnostics and detailed output messages.
- **Editing help**: completion, hover, and go-to-definition.
- **Proof workflows**: Proof Mode with a live Proof State sidebar.
- **Extra tooling**: Visual Editor for selected proof exploration flows.

This walkthrough is intentionally short: it helps you understand the extension surface and set the most important preferences before deeper tutorials.
