# Open NiceGeo settings

Use this step to set up the extension once, in a structured way.

## Settings categories

### 1) Execution
- `nicegeo.execution.mode`
- Choose `dune` for project-local workflows or `installedBinary` for global installs.

### 2) Diagnostics
- `nicegeo.diagnostics.trigger`
- `nicegeo.diagnostics.debounceMs`
- Decide how proactive diagnostics should be while typing.

### 3) Proof State behavior
- `nicegeo.proofState.followCursor`
- `nicegeo.proofState.followCursorDebounceMs`
- Control how frequently proof-state refreshes as you move the caret.

The extension is designed to run mostly automatically after this setup.
